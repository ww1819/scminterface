获取全部数据（按业务顺序）

顺序：科室 → 人员 → 分类 → 供应商 → 生产厂家 → 产品档案 → 出退库单据明细 → 汇总库存 → 明细库存。 供应商/厂商/耗材字典均固定 materialOrDrug=1（仅耗材）；字典每页 50 条翻页；分类不传 keyWord；人员 roleType 0~8 合并。
