获取全部数据（按业务顺序）

顺序：科室 → 人员 → 分类 → 供应商 → 生产厂家 → 产品档案 → 出退库单据明细 → 汇总库存 → 明细库存。 供应商/厂商/耗材字典均固定 materialOrDrug=1（仅耗材）；字典每页 50 条翻页；分类不传 keyWord；人员 roleType 0~8 合并。

出退库时间点击选择，提交格式为 yyyy-MM-dd HH:mm:ss；默认最近 7 天。

分页类接口自动翻页合并；耗时较长请勿重复点击。

一键顺序测试（推荐给实施人员）

各接口表单字段与文档入参一一对应；展开「高级 JSON」可手工覆盖或编辑 POST 请求体。
